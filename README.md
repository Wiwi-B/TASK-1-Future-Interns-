#  Audit de sécurité – OWASP Juice Shop

Ce repository contient les livrables d’un projet de test de sécurité applicative
réalisé sur l’application volontairement vulnérable **OWASP Juice Shop**.

## Objectif du projet
Identifier et documenter des vulnérabilités de sécurité en se basant sur le référentiel
**OWASP Top 10**, à l’aide d’outils de tests de sécurité web.

## Application testée
- Nom : OWASP Juice Shop
- Environnement : Local (Docker)
- Type de test : Black Box

## Outils utilisés
- OWASP ZAP
- Burp Suite (Community Edition)
- Kali Linux
- Docker

## Contenu du repository

- `report/` : Rapport d’audit de sécurité (PDF)
- `owasp-checklist/` : Checklist de conformité OWASP Top 10
- `tool-logs/` : Journaux et preuves issues des outils (ZAP, Burp)
- `video/` : Vidéo de présentation des résultats (optionnel)

## Avertissement
Ce projet est réalisé **uniquement à des fins pédagogiques**.
Les tests ont été effectués sur une application volontairement vulnérable.
